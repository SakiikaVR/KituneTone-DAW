# 狐Tone 2.1.0

狐Tone初の2.1系正式版です。LMMSを基盤としつつ、専用プロジェクト形式とVST3専用ホストを採用したWindows向けDAWとして公開します。

## 主な機能

- VST3インスツルメント／エフェクト専用ホスト
- GUI、パラメーター、MIDI、トラック、アルペジオ／コードを統合した共通画面
- VST3パラメーターの右クリック・オートメーション接続
- MIDIノート、ベロシティ、パン、詳細なピッチベンドの録音・再生
- サンプルトラックの入力デバイス選択とマルチトラック音声録音
- BPM同期アルペジエーターとコードスタッキング
- ピッチを保つタイムストレッチ
- ARA 2の音声ソース、再生領域、テンポ・拍子、状態保存連携
- 常時最大表示のトラックエディターと複数クリップ編集
- 日本語UI、Noto Sans JP、アクセントカラーテーマ
- `.ktpz`／`.ktp`／`.ktt` の狐Tone専用形式

## β版からの主な修正

- VST3ウィンドウとトラック機能を整理し、日本語化を拡充
- 録音したベロシティと複雑なピッチベンドの再生を修正
- VST3パラメーターとオートメーションの接続操作を追加
- アルペジエーターを既定でBPM同期し、選択リストを修正
- 歯車メニュー、終了確認、補助画面・バックグラウンド終了処理を修正
- 非VST3プラグインの登録と旧エフェクト互換復元を廃止
- 標準プロジェクトコンテンツをテンプレートだけに整理
- 全機能説明書を同梱

## ダウンロード

- `KitsuneTone-2.1.0-win64-setup.exe`: インストーラ版
- `KitsuneTone-2.1.0-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.0は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
