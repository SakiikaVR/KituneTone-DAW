# 狐Tone 2.1.3

狐Tone 2.1.3は、ARA 2プラグイン画面の埋め込みを修正した正式パッチ版です。

## 修正内容

- 共通VST3画面を開いた後にARA同期した場合も、ARAエディターを「プラグイン画面」タブ内へ表示
- ARAエディターだけが独立ウィンドウとして開く問題を修正
- ARA再接続後に共通画面へ空のGUI領域が残る問題を修正
- 埋め込み位置、リサイズ通知、閉じる際のビュー解放を維持

## ダウンロード

- `KitsuneTone-2.1.3-win64-setup.exe`: インストーラー版
- `KitsuneTone-2.1.3-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.3は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
