# 狐Tone 2.2.0

パターンエディターを固定スロット方式から、ソングエディターと共通のタイムライン編集方式へ再設計した正式版です。

## 主な変更

- MIDI、サンプル、オートメーションをパターン内へ複数クリップとして配置
- 描画、分割、選択・移動の3編集モード
- 複数選択、コピー／貼り付け、直後へ複製、削除、ミュート、色、左右リサイズ
- ソングエディターと共通のスナップ設定、25～400%横ズーム、`Ctrl+マウスホイール`、横スクロール
- パターンごとの小節数設定（1～999小節）
- 設定した小節数を着色範囲で表示し、パターン内の独立ループ操作を廃止
- ソング／パターンエディターのサンプルトラック間で音声素材をドラッグ移動
- 設定した小節数をパターン再生範囲とトラックエディター上の標準クリップ長へ反映
- MIDI、サンプル、オートメーションの所属パターンと相対位置を直接保存する新形式
- トラックエディター上のパターンプレビューで複数MIDIクリップを正しい位置に表示
- パターン関連ツールバーを日本語化

## プロジェクトについて

2.2.0ではパターンの内部形式を刷新しています。2.1.x以前のプロジェクトとのパターン互換はありません。2.2.0以降で新規プロジェクトを作成してください。

## 配布物

- `KitsuneTone-2.2.0-win64-setup.exe`: インストーラ版
- `KitsuneTone-2.2.0-win64-portable.zip`: ポータブル版

狐ToneはLMMSを基盤とするGPL-2.0-or-laterの派生ソフトウェアです。配布物にはライセンス、帰属表示、対応ソースへの案内を収録しています。
