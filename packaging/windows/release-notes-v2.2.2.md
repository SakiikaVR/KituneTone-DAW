# 狐Tone 2.2.2

作曲中の長時間運用と終了時の安定性を重点的に改善した正式版です。

## 主な修正

- 音声ワーカースレッドの待機・終了競合を修正し、終了時に実行中スレッドを残さないようにしました。
- トラック削除時、再生待ちノートや親子ノートを安全な順序で解放するようにしました。
- VST3の読み込み・処理・終了境界を強化し、処理に失敗したプラグインを安全に停止できるようにしました。
- VST3 MIDIイベント数を制限し、異常な出力によるメモリ消費を抑制しました。
- 内部MIDIルーティングの循環を遮断し、フィードバックによる停止を防ぐようにしました。
- Windows MIDIデバイスの接続・切断とコールバック処理の競合を修正しました。
- 楽器の遅延読み込み、トラック／クリップの遅延UI更新、パターントラック削除時の寿命管理を修正しました。
- 不正なノート番号、ステップ数、選択範囲を受け取った場合の境界チェックを追加しました。
- GUI、切り離した子ウィンドウ、音声・MIDIエンジンを安全な順序で終了するようにしました。
- 復元キャンセルや変換専用モードを含むすべての終了経路で後処理を行うようにしました。

## 配布物

- `KitsuneTone-2.2.2-win64-setup.exe`: インストーラ版
- `KitsuneTone-2.2.2-win64-portable.zip`: ポータブル版
