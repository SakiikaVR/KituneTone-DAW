# 狐Tone 2.2.1

狐Tone 2.2.0のパターン編集とVST3 MIDIルーティングを改善する正式バグ修正版です。

## 修正内容

- パターンエディターの横ズーム変更時に、MIDI・サンプル・オートメーションクリップの幅と横位置が表示倍率へ追従しない問題を修正
- ズームスライダーと`Ctrl+マウスホイール`のどちらでもクリップ表示を即時更新
- MIDI出力バスを持つVST3ジェネレーター／プロセッサーを内部MIDIルーティングした際、トラック再生で変換前と変換後のノートが同時に鳴る問題を修正
- VST3が生成したMIDI出力を優先し、変換前ノートの内部MIDIへの二重送信を停止

## プロジェクトについて

プロジェクト形式は2.2.0と同じです。2.1.x以前のプロジェクトとのパターン互換はありません。

## 配布物

- `KitsuneTone-2.2.1-win64-setup.exe`: インストーラ版
- `KitsuneTone-2.2.1-win64-portable.zip`: ポータブル版

狐ToneはLMMSを基盤とするGPL-2.0-or-laterの派生ソフトウェアです。配布物にはライセンス、帰属表示、対応ソースへの案内を収録しています。
