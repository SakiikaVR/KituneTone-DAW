# 狐Tone 2.1.1

狐Tone 2.1.1は、Windows配布パッケージの表示素材不足を修正した正式パッチ版です。

## 修正内容

- インストーラー版に標準テーマ、アイコン、背景、フォント、日本語翻訳を正しく同梱
- ポータブル版に同じ表示素材を正しく同梱
- 配布物のテーマ素材が欠落した場合に発生していた、灰色の画面、白い四角形のボタン、空欄の設定アイコンを解消
- 配布物にはVST3ホストに必要なプラグインと標準テンプレートのみを収録

## ダウンロード

- `KitsuneTone-2.1.1-win64-setup.exe`: インストーラー版
- `KitsuneTone-2.1.1-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.1は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
