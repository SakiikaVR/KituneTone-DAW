# 狐Tone 2.1.4

狐Tone 2.1.4は、サンプルトラックの空クリップ生成を修正した正式パッチ版です。

## 修正内容

- サンプルトラックの空白部分をクリックしただけでは空クリップを作成しないよう変更
- 音声ファイルのドラッグ＆ドロップによるサンプル追加を維持
- サンプルブラウザーからの追加と音声録音によるクリップ作成を維持
- MIDI、パターン、オートメーションなど、空クリップを編集に使用するトラックの操作は変更なし

## ダウンロード

- `KitsuneTone-2.1.4-win64-setup.exe`: インストーラー版
- `KitsuneTone-2.1.4-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.4は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
