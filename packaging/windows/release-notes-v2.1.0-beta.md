# 狐Tone 2.1.0-beta

LMMSを基盤に、狐Toneとして独自のVST3中心DAWへリブランディングしたベータ版です。

## 主な変更

- VST3インスツルメントとエフェクトのウィンドウを共通化し、GUI・パラメーター・MIDI・トラック機能をタブ化
- トラックエディターをワークスペース内の常時最大表示に固定
- トラックの歯車メニューが安定して開くよう修正
- LCD数値の背景と前景を同じNotoフォントで描画
- 狐Tone専用プロジェクト形式 `.ktp` / `.ktpz` / `.ktt` と `kitsunetone-project` ルートを導入
- LMMSプロジェクト形式を互換ファイルとして開かないよう分離
- 日本語翻訳カタログの未翻訳指定を解消
- VST3・ARAリソースの終了順序を修正し、終了後にプロセスが残る問題を改善
- Qtランタイムを同梱し、`Qt6Widgets.dll` 不足による起動失敗を解消
- インストーラ版とポータブル版を追加

## SmartScreenについて

このベータ版はコード署名証明書が用意されていないため、バイナリは未署名です。Windows SmartScreenが警告を表示する場合があります。ダウンロード後は同梱の `SHA256SUMS.txt` でファイルを確認してください。
