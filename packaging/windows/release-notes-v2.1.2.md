# 狐Tone 2.1.2

狐Tone 2.1.2は、設定ファイルの初回保存とポータブル動作を修正した正式パッチ版です。

## 修正内容

- インストーラー版の初回起動時に設定保存フォルダーを自動作成
- 設定保存時に親フォルダーがない場合は自動作成
- ポータブル版へ判定ファイルを同梱し、設定と作業フォルダーを展開先へ保存
- インストーラー版にはポータブル判定ファイルが混入しないよう分離
- 2.1.1で修正した標準テーマ、アイコン、背景、フォント、日本語翻訳の同梱を維持

## ダウンロード

- `KitsuneTone-2.1.2-win64-setup.exe`: インストーラー版
- `KitsuneTone-2.1.2-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.2は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
