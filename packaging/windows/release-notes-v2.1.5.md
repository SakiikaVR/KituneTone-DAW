# 狐Tone 2.1.5

狐Tone 2.1.5は、パターンエディターのサンプル配置とトラック操作メニューの応答性を修正した正式パッチ版です。

## 修正内容

- パターンエディターのサンプルトラックへ、サンプルブラウザーまたはエクスプローラーから音声素材をドラッグ＆ドロップできない問題を修正
- ドロップした素材を現在のパターンに対応するクリップへ正しく割り当て、パターン長と表示を即時更新
- サンプルトラックをクリックしただけでは空クリップを作らない仕様を維持
- パターンの再生位置、ループ長、サンプル波形の表示幅を音声素材の実時間に一致させるよう修正
- パターン内の同じサンプルトラックへ複数の音声素材を時間位置付きで配置できる機能を追加
- パターン内のサンプルクリップを選択し、左右または上／下のサンプルトラックへドラッグ移動できる操作を追加
- 複数小節のパターンで横スクロールしても、タイムライン、再生位置、クリップ表示が一致するよう修正
- 歯車のトラック操作メニューを開くたびに録音デバイスを同期検索していた処理を分離し、メニューが反応しない、または一時停止する問題を修正

## ダウンロード

- `KitsuneTone-2.1.5-win64-setup.exe`: インストーラー版
- `KitsuneTone-2.1.5-win64-portable.zip`: ポータブル版

## 互換性

狐Tone 2.1.5は64-bit Windows 10／11および64-bit VST3専用です。LMMSの `.mmp`、`.mmpz`、`.mpt`、LADSPA、LV2、VST2、LMMS内蔵プラグインとの互換性はありません。
